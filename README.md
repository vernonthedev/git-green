# Git Green

With **git-green**, you can make your profile look like you've been hard at work... even if you haven't.
NodeJs script to make commits to the past (or the future) to go green on GitHub.

## About

**git-green** helps you create commits on your GitHub profile for any date in the past. Whether you want to fill up your contribution graph or even make cool patterns and artwork.

## Getting Started

Follow these steps to bring your contribution graph to life:

**Clone this repository**

```bash
git clone https://github.com/vernonthedev/git-green.git
cd git-green
pnpm install
pnpm beautiful
```
